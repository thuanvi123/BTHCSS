Jquery callback

Callback function là 1 hàm được thực thi sau khi 1 sự kiện hay 1 hiệu ứng được thực thi và kết thúc hoàn toàn

Ví dụ về callback như sự kiện các bạn gọi điện hay gửi tin nhắn xong nhà mạng sẽ trừ tiền vào tài khoản vậy

hành động trừ tiền của nhà mạng viễn thông như 1 callback được thực hiện ngay sau khi các bạn gửi sms hay gọi điện

Callback trong jquery rất hay gặp ở dạng 1 function không có tên ví dụ như :

    function() {
        // code thực thi trong callback
    }

Ví dụ như document ready cũng có chứa 1 callback

    $(document).ready(function() {
        // code chạy khi document đã được tải xuống
    });

Hay 1 sự kiện click chuột

    $("#btn").click(function() {
        // code chạy trong callback sau khi nút có id là btn được click
    });
